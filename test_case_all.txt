테스트 케이스 1

행동: 입력칸에 "일본어 공부" 입력 후 엔터
예상 결과: 목록에 추가됨
실제 결과: 추가됨
결과: ✅ 통과

테스트 케이스 2

행동: 체크박스 클릭
예상 결과: 글자에 줄이 그어짐
실제 결과: 글자에 줄이 그어짐
결과: ✅ 통과

테스트 케이스 3

행동: X버튼 클릭
예상 결과: 해당 항목이 목록에서 사라짐
실제 결과: 해당 항목이 목록에서 사라짐
결과: ✅ 통과

테스트 케이스 5
* 행동: 3개의 할 일 중 두번째 할 일 체크 이후 Active 클릭
* 예상 결과: 체크했던 할 일이 사라짐
* 실제 결과: 체크했던 할 일이 사라짐
* 결과: ✅ 통과

테스트 케이스 6
* 행동: Completed 클릭
* 예상 결과: 첫번째와 세번째 할 일이 사라지고 완료했던 할 일 표시
* 실제 결과: 첫번째와 세번째 할 일이 사라지고 완료했던 할 일 표시
* 결과: ✅ 통과

테스트 케이스 7

행동: 할 일 5개 추가
예상 결과: 5개 전부 목록에 표시
실제 결과: 5개 전부 입력 순서대로 목록에 표시됨
결과:✅ 통과
우선순위: 높음
